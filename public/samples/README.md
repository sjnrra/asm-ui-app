# 依存ファイルディレクトリ

このディレクトリに配置されたファイルは、アプリ起動時に自動的に読み込まれます。

## 使用方法

1. 依存関係にあるファイル（.INC、.MAC、.ASMなど）をこのディレクトリに配置
2. `dependencies.json` にファイル名を追加

例：
```json
{
  "files": [
    "REGS.INC",
    "SAVEMAC.MAC",
    "MYMACRO.MAC"
  ]
}
```

3. アプリを再起動すると、自動的に読み込まれます

## ファイル名の指定

- ファイル名は大文字・小文字を区別しません
- 拡張子は自動的に検索されます（例：`COPY REGS` は `REGS.INC` を検索）
- `COPY` 文で参照する際は、拡張子なしでファイル名を指定できます
